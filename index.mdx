    Quick start

  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get started with Arre in minutes
  </p>

  <Steps>
    <Step title="Clone the repository">
      Clone the Arre repository to your local machine:

      ```bash
      git clone https://github.com/pedro3087/arre.git
      cd arre
      ```
    </Step>

    <Step title="Install dependencies">
      Install all required dependencies using npm:

      ```bash
      npm install
      ```
    </Step>

    <Step title="Start the development server">
      Run the development server and open Arre in your browser:

      ```bash
      npm run dev
      ```

      The app will be available at `http://localhost:5173`
    </Step>

    <Step title="Sign in with Google">
      Click the **Continue with Google** button on the login page to create your account and start managing tasks.
    </Step>
  </Steps>
</div>
